# RunRunRun Web Frontend (Shareable URL)

This is a browser frontend you can deploy to Vercel and share with friends.

## Deploy to Vercel

1. Push repo to GitHub.
2. Go to https://vercel.com/new
3. Import your repo.
4. Set **Root Directory** to `web`
5. Framework preset: **Other**
6. Deploy.

You will get a URL like:
`https://runrunrun-web.vercel.app`

## First use

1. Open your Vercel URL.
2. Enter backend URL (Render URL) in **Backend Base URL** field.
3. Register/Login.
4. Start activity, load activities, stop selected, like/comment.

## Backend URL example

`https://your-render-backend.onrender.com`
